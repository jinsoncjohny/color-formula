Color Formula

Color Formula is a framework to standardize the color extraction process. It uses a token-based approach, similar to color tokens, but instead of mapping to a fixed value, each token is a rule that pulls the right color from any image.
Color Formula tackles and standardizes these challenges: Unified language
Define a token once and have it mean the same thing everywhere.Extracting colors like dominant, vibrant, muted, or contrast from any image consistently across platforms. This site will serve as an archive of every formula token and the logic behind its extraction.
Reusable libraries & plugins
Build tools around one common concept to streamline the design-to-development workflow.
Community-driven archive
Share and refine formulas together, growing an open library of extraction rules.
